## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
3. Deploy expo project
   ```npm install --global eas-cli 
      eas login
      eas init --id "your_id_project in expo"
      eas build --platform android
      eas build -p android --profile preview
   ```
